# Beam Livestreaming Platform - Logo Assets

This directory contains the logo and branding assets for the Beam Livestreaming Platform.

## Logo Variations

### 1. **logo.svg** - Main Logo
- **Dimensions**: 200×60px
- **Usage**: Primary logo for light backgrounds
- **Features**: Animated live indicators, gradient beam effect

### 2. **logo-horizontal.svg** - Horizontal Layout
- **Dimensions**: 300×80px  
- **Usage**: Wide format applications (headers, banners)
- **Features**: Full platform name, larger format

### 3. **logo-dark.svg** - Dark Theme Version
- **Dimensions**: 200×60px
- **Usage**: Dark backgrounds and themes
- **Features**: Glow effects, light text on dark background

### 4. **icon.svg** - Standalone Icon
- **Dimensions**: 60×60px
- **Usage**: App icons, profile pictures, compact spaces
- **Features**: Icon only without text

### 5. **favicon.svg** - Web Favicon
- **Dimensions**: 32×32px
- **Usage**: Browser tabs, bookmarks
- **Features**: Simplified design optimized for small sizes

## Design Elements

### Color Palette
- **Primary Gradient**: #FF6B6B → #4ECDC4 → #45B7D1
- **Text Dark**: #2C3E50 → #34495E
- **Text Light**: #FFFFFF → #ECF0F1
- **Live Indicator**: #FF4757
- **Secondary Text**: #7F8C8D / #BDC3C7

### Typography
- **Font Family**: Arial, sans-serif
- **Main Title**: Bold, 18-28px
- **Subtitle**: Regular, 10-14px with letter spacing

### Animation
All logos feature subtle pulsing animation on the "live" indicator dots to emphasize the livestreaming nature of the platform.

## Usage Guidelines

1. **Maintain aspect ratios** when scaling
2. **Use appropriate version** for background (light/dark)
3. **Ensure minimum size** of 60px width for readability
4. **Preserve animations** in web implementations
5. **Use vector format** (SVG) for best quality at all sizes

## File Formats
All assets are provided in SVG format for:
- ✅ Scalability without quality loss
- ✅ Small file sizes
- ✅ Animation support
- ✅ Easy customization
- ✅ Web-ready format

## Implementation Example

```html
<!-- Web favicon -->
<link rel="icon" type="image/svg+xml" href="/assets/favicon.svg">

<!-- Main logo in HTML -->
<img src="/assets/logo.svg" alt="Beam Livestreaming Platform" height="60">

<!-- Dark theme logo -->
<img src="/assets/logo-dark.svg" alt="Beam Livestreaming Platform" height="60" class="dark-theme">
```

## Brand Concept

The logo represents:
- **Beam**: Signal transmission, broadcasting, connection
- **Gradient Colors**: Modern, dynamic, engaging
- **Animation**: Live, real-time, active streaming
- **Geometric Design**: Technical precision, reliability
- **Professional Typography**: Trust, clarity, readability
